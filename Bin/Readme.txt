Windows10下直接解压到任意目录运行X330Backlight.exe，第一次运行会显示设置程序，按照需要进行设置即可。
Windows7下请先安装.NET Framework 4.6.1, 下载地址：https://www.microsoft.com/zh-CN/download/details.aspx?id=49982

更新历史
2018-08-04 V1.0.1
1. 新增播放电影或听音乐时不进入省电模式，需要多媒体播放器支持向系统申请禁止进入省电或待机模式。
2. 修复一处语言翻译错误问题。

2018-08-03 V1.0 Beta4
1. 增加图形化配置界面，第一次启动的时候会显示配置界面，双击托盘图标可以显示配置界面。
2. 修复快捷键调光和电源键调光不同步的问题。
3. 托盘图标增加右键菜单，可以手动退出程序。

2018-08-02 V1.0 Beta3
1. 增加一种圆形OSD界面。
2. 支持系统长时间不使用自动降低背光亮度，默认值插电源为5分钟，不插电源为3分钟。
3. 支持自定义托盘图标，目前自带4种托盘图标。

2018-08-01 V1.0 Beta2
1. 修复有时休眠后再唤醒调光不起作用的问题。
2. 支持多国语言，可以自己修改或者添加语言，语言文件在EXE目录下的Languages文件夹中。
3. 支持使用ThinkVantage键进行一键开关屏幕，默认为开启状态。

2018-07-31 V1.0 Beta1
1. 使用WPF重写程序，OSD菜单更加平滑，完美匹配Win10，Win7需安装.NET Framework4.6.1。
2. 使用注册表监控的方式实现Fn+F8,F9调光，系统占用资源低。
3. 支持关盖熄屏，对于一些兼容性有问题的外接显示器，可设置成关盖只将屏幕亮度调节到最低。
4. 支持显示右下角图标，图标可显示当前背光亮度。
5. 支持开机自动启动。
